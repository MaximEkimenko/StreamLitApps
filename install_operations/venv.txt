#### 3.  Виртуальное окружение
- из целевой папки создание:
```cmd
cd "целевая папка"
python -m venv имя_окружения (обычно venv)
```
- активация из целевой папки
```cmd
.venv\Scripts\activate
```
- далее можно пользоваться pip для этого окружения и запускать python со своим окружением из \venv\Scripts:
```cmd
"C:\python\OTK BOT\venv\Scripts\python.exe" omzit_otk_bot.py
```
- если появляется ошибка pip, то необходимо обновить pip
```cmd 
python -m pip install --upgrade pip
